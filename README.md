## Prerequisites

- A fresh linux installation (one of the following):
    - Arch Linux - Hyprland
    - Fedora - Cosmic
- Internet connection
- sudo privileges

## Installation

1. Clone this repository:

```bash
git clone https://github.com/pszponder/crucible.git
```

2. Navigate inside the cloned directory and run the setup script:

```bash
./run.sh
```
